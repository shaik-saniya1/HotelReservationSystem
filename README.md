# Hotel Reservation System (Java Swing GUI)

## 📌 Project Description
A desktop-based Hotel Reservation System developed using Java.  
The application allows users to search rooms, book rooms, cancel bookings, and view reservations using a graphical user interface.

## 🚀 Features
- Search Rooms by Type
- Book Room with Date Validation
- Cancel Booking
- View All Bookings
- File-Based Data Storage (rooms.txt, bookings.txt)
- User-Friendly Swing GUI

## 🛠 Technologies Used
- Java
- Swing (GUI)
- File Handling
- Object-Oriented Programming (OOP)
- Git & GitHub

## ▶ How to Run

1. Compile all Java files:
   javac *.java

2. Run the GUI:
   java HotelGUI

## 📂 Project Structure
- Room.java
- Booking.java
- HotelService.java
- Storage.java
- HotelGUI.java
- Main.java
- rooms.txt
- bookings.txt
